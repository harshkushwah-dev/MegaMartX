## Installation

```bash
git clone https://github.com/harshkushwah-dev/MegaMartX.git
cd MegaMartX
npm install
npm run dev
```

## Usage

- Visit `http://localhost:5173` in your browser.
- Register/Login to explore features.
- Add products to cart, wishlist, and checkout.
